# Changelog

## Phase 2 - 2025-12-30

### 🎉 새로운 기능

#### 런 템플릿
- 자주 사용하는 케이스 조합을 템플릿으로 저장
- 템플릿에서 원클릭으로 런 생성
- 공개/비공개 템플릿 지원
- 템플릿 관리 (수정/삭제)

#### 키보드 핫키 고도화
- 10개 이상의 단축키 지원
- `p`: 이전 미실행 케이스
- `g`: 첫 케이스로 이동
- `G`: 마지막 케이스로 이동
- `c`: 코멘트 입력 포커스
- `r`: 통계 새로고침
- `Esc`: 코멘트 닫기
- 단축키 힌트 자동 숨김 (5초 후)

#### 케이스 중복 감지
- 케이스 제목 입력 시 실시간 유사도 검사
- 정확 일치, 부분 문자열, 단어 기반 유사도 계산
- 유사도 50% 이상 시 경고 표시
- 상위 5개 유사 케이스 표시

#### 운영 대시보드
- **자주 실패하는 케이스**: 최근 30일 내 실패 횟수 Top 10
- **오래된 케이스**: 90일 이상 미수정 케이스
- **불안정한 케이스 (Flaky)**: Pass/Fail이 불규칙한 케이스
- 실시간 분석 데이터 로드

#### 섹션 관리 개선
- 드래그앤드롭으로 섹션 순서 변경
- 드래그앤드롭으로 계층 구조 변경
- 호버 시 편집 버튼 표시
- 하위 섹션 추가 기능

#### 런 생성 UX 개선
- 케이스 선택 모달 UI
- 섹션별 필터링
- 체크박스 선택
- 전체 선택 기능
- 실시간 선택 카운트

### 🔧 개선사항
- 키보드 단축키 충돌 방지
- 입력 필드에서 단축키 무시
- 드래그 중 시각적 피드백
- 모달 외부 클릭 시 닫기

### 📊 데이터 모델
- `RunTemplate` 테이블 추가
- 템플릿 관련 API 엔드포인트 추가
- Analytics API 엔드포인트 추가

## Phase 1 - 2025-12-30

### 🎉 초기 출시

#### 인증 및 사용자 관리
- 로그인/로그아웃
- 회원가입
- 3가지 역할 (admin, author, runner)

#### 프로젝트 관리
- 프로젝트 생성/조회
- 프로젝트별 케이스 관리

#### 섹션 트리
- 계층적 섹션 구조
- 섹션 생성/수정/삭제
- 섹션별 케이스 필터링

#### 테스트 케이스
- Quick Add (제목만으로 즉시 생성)
- 인라인 편집
- 태그 시스템
- 우선순위 (P0-P3)
- 검색 및 필터
- 아카이브

#### 테스트 런
- 런 생성 (케이스 선택)
- 런 실행
- 키보드 단축키 (1-5, n, j/k)
- 결과 기록 (Pass/Fail/Blocked/Retest/NA)
- 코멘트 및 첨부파일

#### 리포트
- 런 진행률
- Pass rate
- 상태별 통계
- 시각적 진행률 바

#### UI/UX
- 2-pane 레이아웃
- HTMX 기반 부분 업데이트
- 반응형 디자인
- 플래시 메시지
